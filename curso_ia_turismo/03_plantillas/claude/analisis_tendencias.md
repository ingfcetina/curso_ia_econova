## Métricas
| Métrica | Descripción | Fórmula | Objetivo |
|---------|-------------|---------|----------|
| Adopción | Tasa | Usuarios / Total | >30% |
| Impacto | Efecto | Beneficio / Inversión | >2x |
| Eficiencia | Mejora | (Actual - Anterior) / Anterior | >20% |
| Satisfacción | NPS | Promotores - Detractores | >50 |
| ROI | Retorno | (Beneficio - Inversión) / Inversión | >100% |

## Plantillas de Prompt

### 1. Identificación de Tendencias
```
Analiza la tendencia:
Categoría: [CATEGORIA]
Tema: [TEMA]
Fuente: [FUENTE]

Proporciona:
- Descripción de la tendencia
- Factores impulsores
- Potencial impacto
- Horizonte temporal
- Nivel de madurez
```

### 2. Análisis de Mercado
```
Evalúa el mercado para:
Tendencia: [TEMA]
Segmento: [SEGMENTO]
Período: [PERIODO]

Identifica:
- Nivel de adopción
- Demanda potencial
- Oferta actual
- Competencia
- Regulaciones
```

### 3. Análisis de Impacto
```
Analiza el impacto de:
Tendencia: [TEMA]
Área: [AREA]
Horizonte: [HORIZONTE]

Evalúa:
- Efecto en operaciones
- Impacto en clientes
- Cambios en negocio
- Requisitos técnicos
- Necesidades de recursos
```

### 4. Plan de Adaptación
```
Desarrolla un plan para:
Tendencia: [TEMA]
Objetivo: [OBJETIVO]
Timeline: [TIMELINE]

Incluye:
- Estrategia de implementación
- Acciones específicas
- Recursos necesarios
- Cronograma
- Métricas de éxito
```
