# Requisitos del Curso de IA para Agencias de Viajes

## Requisitos Técnicos

### Hardware
- Computadora portátil con:
  - Procesador: Intel i5/Ryzen 5 o superior
  - RAM: 8GB mínimo (16GB recomendado)
  - Almacenamiento: 256GB mínimo
  - Conexión a internet estable (mínimo 10Mbps)

### Software
- Sistema operativo actualizado (Windows 10/11, macOS 10.15+, o Linux)
- Navegador web actualizado (Chrome, Firefox, o Edge)
- Microsoft Office o Google Workspace
- Editor de texto (VS Code, Sublime Text, o similar)
- Git (opcional, para versionamiento)

## Cuentas y Accesos Necesarios

### Herramientas de IA
1. **ChatGPT**
   - Cuenta en OpenAI (ChatGPT Plus recomendado)
   - Acceso a la API (opcional)

2. **DeepSeek**
   - Cuenta de desarrollador
   - Acceso a la API

3. **Claude**
   - Cuenta en Anthropic
   - Acceso a la API (opcional)

### Herramientas de Marketing Digital
1. **Google Workspace**
   - Cuenta de Google
   - Acceso a Google Analytics
   - Google Search Console

2. **Herramientas de Social Media**
   - Cuentas en plataformas principales (Facebook, Instagram, LinkedIn)
   - Herramientas de gestión de redes sociales (opcional)

3. **Herramientas de Análisis**
   - Google Analytics
   - Herramientas de análisis de competencia (opcional)

## Recursos Adicionales

### Bibliografía Recomendada
- "Artificial Intelligence in Marketing" - O'Reilly
- "The Age of AI" - Henry Kissinger, Eric Schmidt
- "Marketing 5.0" - Philip Kotler

### Enlaces Útiles
- Documentación oficial de las APIs
- Comunidades de desarrolladores
- Foros de marketing digital

## Preparación Previa
1. Crear cuentas en todas las plataformas mencionadas
2. Familiarizarse con las interfaces básicas
3. Revisar la documentación introductoria
4. Preparar ejemplos de casos de uso específicos de la agencia

## Notas Importantes
- Se proporcionarán credenciales temporales para herramientas premium durante el curso
- Se recomienda tener un segundo dispositivo para tomar notas
- Mantener actualizadas todas las herramientas y extensiones
- Tener un respaldo de la información importante
